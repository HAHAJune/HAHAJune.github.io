Git常用命令

Posted on 2019-09-25

记录下平时用的比较多的Git命令

初始工程

help 帮助

clone 克隆仓库

init 创建或者重新修改仓库

config 仓库配置

当前分支工作

add 添加文件到暂存区（追踪文件）

mv 移动或重命名文件

reset 重置 HEAD 指向

rm 从工作区和暂存区移除文件

查看工作历史状态

bisect 二分法查找哪一次代码提交引入了错

grep 从提交历史或者工作目录中检索关键字

log 查看提交日志

show 显示各种类型的对象

status 显示工作区状态

切换合并对比标记分支

branch 分支的增删改查

checkout 切换分支或者或者还原工作区文件

commit 记录到仓库的提交

diff 对比不同

merge 合并

rebase 变基，把一段提交记录放在另一段提交记录的头部

tag 标签

stash 储藏

clean 移除工作目录中未被追踪的文件

合作

fetch 从另一个存储库下载对象和引用

pull 从另一个存储库或本地分支获取并集成(整合)

push 将本地分支的更新，推送到远程主机

提交恢复

恢复提交
